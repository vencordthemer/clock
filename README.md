# Simple Clock App

A basic web-based clock application built with HTML, CSS, and JavaScript.

## Features

*   Displays the current time (HH:MM:SS).
*   Updates every second.
*   Allows toggling between 12-hour (with AM/PM) and 24-hour time formats using a button.
*   Clean, minimalist design.

## How to Run

1.  Clone or download this repository.
2.  Open the `index.html` file in your web browser.

## Technologies Used

*   HTML 5
*   CSS 3
*   Vanilla JavaScript
